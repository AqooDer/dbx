| PostgresSQL | mysql       |

# 支持的数据处理模型

详情可以查看 **测试模型** 文档

| 模型    | 描述                                |
|-------|-----------------------------------|
| 表拷贝   | 支持将源表直接拷贝为目标表                     |
| 表名称修改 | 支持将表名称修改后数据拷贝                     |
| 表字段删减 | 支持将表数字段删减后数据拷贝                    |
| 表字段增加 | 支持直接在目标表上添加字段，然后拷贝数据。 添加字段支持默认值设置 |
| 表字段修改 | 支持字段修改，甚至支持表字段的类型修改               |
| 表合并   | 支持将多个源表的数据合并到一个目标表，支持条件合并         |
| 表拆分   | 支持将单表的数据拆分到多个表数据中，支持条件拆分          |
